About Me

For the last year I have been a part of the Human-Centered Data Science Lab. I have focused on building interactive interactive visualisations to help make sense of the big data that we are working with. The big data we are using is stored on a server that I have used SQL through SQLAlchemy on a remote connenction to query the data I needed. I use my strong pandas skills too wrangle that data into formats sutable for visualisation or statistical analysis. For most visuzlisation I use altair a python library that can create visualisations that can be displayed on any webpage. I also am experienced with Matplotlib, Plotly, Seaborn , ggplot in R, Vege-Lite in JS, and Tableau.

I love learning new things and have built up a diverse set of skills including building electronics and moble app development. I like to pick up new libraries and incrperate them into my projects. I don't shy away from reading and modifying the exisitng libraries.
